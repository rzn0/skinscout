# Privacy Policy for Skin Scout

**Effective date:** 18-05-2026
**Last updated:** 20-05-2026
**Publisher:** Mvlfunction
**Contact:** mvlfunx@gmail.com

Skin Scout ("the Extension") helps users compare Steam Community Market item prices with external marketplaces. This Privacy Policy explains what data is processed and how.

## 1. What We Process

### A) Website content (Steam listing context)

When active on Steam Community Market listing pages, the Extension reads limited page data needed to perform comparisons, such as:

- Item/listing identifier (for example, market hash name)
- Displayed pricing context
- Listing URL/page context

### B) Local extension data

The Extension stores data locally in your browser using `chrome.storage`:

- User preferences (for example: currency, thresholds, UI settings)
- Short-lived cached comparison results

### C) Network request metadata

When requesting market/FX data, your browser may send technical request data (such as IP address and user agent) directly to third-party services as part of normal web requests.

## 2. How Data Is Used

Data is used only to:

- Fetch and display marketplace comparison results
- Calculate estimated savings/liquidity signals
- Apply your local settings
- Reduce request volume through local caching

## 3. Data Sharing and Transfers

We do **not** run a backend that collects your extension data centrally.

However, the Extension sends comparison queries directly from your browser to:

- `steamcommunity.com`
- `skinbaron.de`
- `api.skinport.com`
- `api.dmarket.com`
- `open.er-api.com` (exchange rates)

If you click outbound marketplace links, those sites receive your request directly. Some links may include affiliate parameters.

## 4. What We Do Not Do

We do **not**:

- Sell personal data
- Collect account credentials, payment card data, or private messages
- Track full browsing history
- Run remote code from external servers (all executable extension code is packaged with the extension)

## 5. Permissions Used

- **Host permissions:** Used to request marketplace and FX data required for comparisons
- **Storage:** Used to save local preferences and cache results
- **Tabs:** Used to detect/analyze the active Steam listing tab when triggered by the user

## 6. Data Retention

- Preference and cache data are stored locally on your device
- Cached data is temporary and periodically refreshed
- You can remove stored data by clearing extension data or uninstalling the Extension

## 7. Security

We use reasonable measures to minimize data use and keep processing limited to what is necessary for extension functionality. No system can be guaranteed 100% secure.

## 8. Third-Party Services

Marketplace and FX providers operate under their own privacy policies and terms. Please review those policies before using their services.

## 9. Children's Privacy

The Extension is not directed to children under 13 (or the minimum age required in your jurisdiction).
